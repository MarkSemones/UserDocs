Identity Provider Set Up for Facebook

Internal note: Begin in Facebook

D R A F T

Register your application with Facebook and obtain a client ID and client secret.

  1. Go to (http://developers.facebook.com). Note that you may be prompted to register if you have not previously used the Facebook Developer site.

  2. Select Add a New App from the My Apps menu.

  3. In the Platform Selection screen, select Website.

  4. When prompted, enter a descriptive name and select Create New Facebook App ID.

  5. In the confirmation dialog, select a category, then click the Create New Facebook App ID button.

  6. Copy the following values from your Facebook Developer App Settings page:

    • The Facebook App ID. You will paste this value in the Client ID field in the Okta Identity Provider configuration page.

    • The Facebook App Secret. You will paste this value to the Client Secret field In the Okta Identity Provider configuration page. Note that you need to click Show on the Facebook page to obtain the Client Secret.

Internal note: Configure settings in the Okta Admin Dashboard

  1. In a separate browser or tab, go to the Okta Admin dashboard. [For what to do in Okta, we'll provide a link to the Service Cloud doc.]

Internal note: Return to Facebook

  1. On the Facebook Developer App Settings page, click the Add Platform button, then select Website from the pop-up.

  2. Paste the Okta Identity Provider Login URL that you copied from the Okta Idenity Provider page into the field labeled URL of your site.

    The Okta URL you paste into the field should look similar to this: "https://example.okta.com/sso/authorize/0ab1c2de3fGHIjKLM4n5"

  3. Click Save Changes .
