How to Configure SAML 2.0 for IntraLinks

Please sign-in to the Okta Admin app to have your organization specific variables generated for you. You might not be able to complete the setup without these generated variables.

  1. Contact IntraLinks at support@intralinks.com and request that they enable SAML.

  2. Attach the following metadata in an .xml file to the email.

    Sign in to Okta Admin app to have this variable generated for you.
  3. IntraLinks will process your request and enable SAML.

  4. After your request is processed by IntraLinks, use the following URLs as need, after substituting the data shown below for <entityID>.

    Production: https://services.intralinks.com/login/?PartnerIdpId=<entityID>
    Sandbox:    https://preview.intralinks.com/login/?PartnerIdpId=<entityID>

    Sign into the Okta Administrator Dashboard to generate this value.

  5. Done!
