How to Configure SAML 2.0 for eLogic Learning

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Contact eLogic Learning Support and request that they enable SAML 2.0 for your account.
  2. Attach the following:

    • Issuer:Sign into the Okta Admin Dashboard to generate this variable.
    • SAML URL:Sign into the Okta Admin Dashboard to generate this variable.
    • Certificate: Sign into the Okta Admin Dashboard to generate this variable.

  3. eLogic Learning will process your request and after receiving a confirmation email, you can start assigning people to the application.
  4. Done!

    Notes:

    SP-initiated and IDP-initiated flows are supported.

    Just in Time (JIT) provisioning is not supported.

    For SP-initiated flows, go to https://<YourBaseURL>/default.aspx
