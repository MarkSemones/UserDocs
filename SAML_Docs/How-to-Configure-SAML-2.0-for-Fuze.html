How to Configure SAML 2.0 for Fuze

Please sign-in to the Okta Admin app to have your organization specific variables generated for you. You might not be able to complete the setup without these generated variables.

  1. Contact Fuze to enable SAML for your Enterprise organization.

  2. Copy the following Metadata into a text file. Save it with the file name metadata.xml. Attach this file to the message.

    Sign into the Okta Admin dashboard to generate this value.
  3. Fuze will process your request and send you a Discovery Key.
  4. In Okta, navigate to the General tab for the Fuze app. Enter the Discovery Key, and then select Save, as shown below.

  5. Done!
