How to Configure SAML 2.0 for SAManage

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

Go to the Setup page in SAManage

  1. Click here to log into SAManage.

  2.  Click on Setup in the upper right corner of the page.

Go to the Single Sign-On edit page

  1. Click the Single Sign-On button to edit SAML settings.
