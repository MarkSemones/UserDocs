How to Configure SAML 2.0 for HelpGizmo

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Log into HelpGizmo as an administrator.
  2. Navigate to Settings > Security.
  3. Select the Enable SAML checkbox.
  4. Enter the following values (see screen capture at end of step for reference):
        • iDP EntityID. Paste the following into this field:Sign in to the Okta Admin app to generate this variable.
        • Login URL. Paste the following into this field: Sign in to the Okta Admin app to generate this variable.
        • Logout URL. Paste the following into this field: Sign in to the Okta Admin app to generate this variable.
        • Download the x.509 Certificate in .cert format, the click Browse and upload the downloaded certificate:
          Sign in to the Okta Admin app to generate this variable.


  5. Select Save.
  6. Copy your instance ID from the address bar in your browser:

  7. In Okta, navigate to the General tab for the HelpGizmo app, then select Edit.
  8. Select Entity ID: Reader Mapping.
  9. Enter the ACS URL: https://app.helpgizmo.com/kb/map-saml/id/<yourID> where yourID is the value from step 6, above.
  10. Select Save.
  11. In HelpGizmo, select the Map Reader Fields link, then enter the following values (see screen shot at end of step for reference):
    • SSO ID: userid
    • Username/Email: userid
    • Custom Field 1: firstname
    • Custom Field 2: lastname


  12. Select Save.
  13. In Okta, navigate to the General tab for the HelpGizmo app, then select Edit.
  14. Select the Entity ID: Knowledge Base.
  15. Enter the ACS URL: https://<YourSubDomain>.helpgizmo.com/help/saml-login where YourSubDomain is your HelpGizmos subdomain.
  16. Select Save.
  17. Done!
Notes:
  • IdP-initiated flows, SP-initiated flows, and Just in Time (JIT) provisioning are supported.
  • For SP-initiated flows, go to the login page of your account (https://<YourSubDomain.HelpGizmo.com) and select the Login link.
