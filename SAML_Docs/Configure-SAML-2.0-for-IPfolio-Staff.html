How to Configure SAML 2.0 for IPfolio - Staff

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

 

  1. Send an email to the IPfolio Support team and request that they enable SAML 2.0 for your account.

  2. Save and attach the following metadata link:

    Sign into the Okta Admin dashboard to generate this value.

  3. The IPfolio Support team will process your request and provide you with Login URL and Entity ID values.

  4. In Okta, select the General tab for the IPfolio app, then click Edit.

    • Enter the Login URL and Entity ID values provided by IPfolio into the corresponding fields.
    • Click Save.
  5. Done!

 

Notes:

SP-initiated flows and IDP-initiated flows are supported.

Just In Time (JIT) Provisioning is not supported.


For SP-initiated Flows

Open your Entity ID value (provided by IPfolio support team) in a browser.
