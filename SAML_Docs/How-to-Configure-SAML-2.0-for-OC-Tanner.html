How to Configure SAML 2.0 for OC Tanner

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Send an email to OC Tanner support at infosec@octanner.com to request a SAML 2.0 integration. The subject of the email message should be: <YourCompanyName> - Okta SAML Integration, where <YourCompanyName> is the company name you used when you registered with OC Tanner.

  2. Copy the following IDP Metadata into a text file. Save it with the file name metadata.xml  and attach it to the message.

    Sign into the Okta Admin dashboard to generate this value.

  3. OC Tanner will notify you when SAML is enabled.
