How to Configure SAML 2.0 for CloudBees

Please sign-in to the Okta Admin app to have your organization specific variables generated for you. You might not be able to complete the setup without these generated variables.

  1. Sign in to CloudBees at https://grandcentral.cloudbees.com/login.
  2. Click the Settings gear icon in the top right corner.
  3. In the window that opens, select SSO Integration in the menu bar on the left, as shown below.
  4. In the SSO Integration window, click Create new SAML 2.0 integration, as shown below.
  5. The Set Up SAML 2.0 Single Sign Window shown below opens.

  6. Copy the following information to the Remote Logon URL field.

    Sign into the Okta Admin dashboard to generate this value.

  7. Copy the following certificate to the Certificate field.Sign in to the Okta Admin app to have this variable generated for you.

  8. Enter your domain name in the Authentication domains field.

  9. If the Provision user box is not checked, click it to enable.

  10. Save the information.
  11. Done!
