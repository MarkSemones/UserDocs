How to Configure SAML 2.0 for Talkdesk

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. In Okta, select the Sign On tab for the Talkdesk app, then make a copy of the Metadata Url, as shown below:

  2. Log in to Talkdesk as an administrator.

  3. Navigate to the Admin > Integrations section, then select the Okta integration:

  4. Select the Configuration button, then enter the Metadata URL you copied from Okta in step 1, then click Save.

  5. Talkdesk will automatically fetch your SSO credentials and save it for your account:

  6. In Okta, select the General tab for the Talkdesk app, then copy and paste the following Entity ID value into the Audience Restriction field:

    Sign into the Okta Admin Dashboard to generate this variable.

  7. Click Save.

  8. Done!

    Notes:

    IdP-initiated flows and SP-initiated flows are supported.

    Just In Time (JIT) provisioning is not supported.
