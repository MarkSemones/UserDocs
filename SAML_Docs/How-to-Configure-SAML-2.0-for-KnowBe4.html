How to Configure SAML 2.0 for KnowBe4

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Contact KnowBe4 Support (https://knowbe4.zendesk.com/hc/en-us/requests/new) and request that they enable SAML 2.0 for your account.

  2. Attach the following:

    • IDP SSO Target Url:

      Sign into the Okta Admin Dashboard to generate this variable.

    • IDP Cert Fingerprint:

      Sign into the Okta Admin Dashboard to generate this variable.

  3. KnowBe4 Support will process your request. After receiving a confirmation email, you can start assigning people to the application.

  4. Done!

    Notes:

    IdP-initiated flows, SP-initiated flows, and Just In Time (JIT) Provisioning are all supported.

    For SP-initiated flows:

    Go to the https://training.knowbe4.com/auth/saml/yourCompanyDomain page:
