Next steps

Wait for Yammer to send you further instructions on how to enable Single Sign-On for your organization. If you want to allow your users to sign-on to Yammer immediately, you can change the sign-on mode to Secure Web Authentication.

You can assign Yammer access to your users at anytime from their user profile. Go to the People page within Okta to view and edit user profiles.
