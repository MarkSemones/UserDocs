How to Configure SAML 2.0 for BlueJeans

Please sign-in to the Okta Admin app to have your organization specific variables generated for you. You might not be able to complete the setup without these generated variables.

  1. Sign in to BlueJeans  at www.bluejeans.com as a user with rights to edit SAML configuration.
  2. Select the Admin tab, and then the Security tab, as shown at the bottom of this page.
  3. Click the SAML Single Sign On button.

  4. Download the following Certificate. Add it to the Certificate Path in Blue Jeans by selecting Browse and uploading it.

    Sign in to the Okta Admin app to generate this variable.

  5. Enter the following text for the Login URL.

    Sign in to the Okta Admin app to generate this variable.

  6. Copy the following URL into the SAML Transfer URL field.

    Sign into the Okta Admin dashboard to generate this value.

  7. Copy the following URL into the Single Logout Service field.

    Sign into the Okta Admin dashboard to generate this value.

  8. Modify the following text for the Password Change URL, where <company name> is the name of your company. For example, if your company is named acme, enter https://acme.bluejeans.com.

    https://<company name>.bluejeans.com

  9. Modify the following text for the Logout URL, where <company name> is the name of your company.

    https://<company name>.bluejeans.com
  10. Fill in the User ID and Email fields with the correct information for your organization.

    Note: If the User ID field is not visible, uncheck Pick User ID from <saml2:NameID> element. This box is checked in the screenshot below.
  11. Copy the information in the RelayState field in BlueJeans. In Okta, paste this information into the Default Relay State field, as shown below.

  12. Click Save Changes.

  13. Done!
