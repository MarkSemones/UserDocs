How to Configure SAML 2.0 for LucidChart

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

    1. Log in to LucidChart as an administrator.

    2. Navigate to Team > App Integration > SAML.

    3. Select Activation.

    4. Check the Enable SAML Integration box.

    5. Enter your Domain.

    6. Copy and paste the following into the Identity Provider Metadata field:

      Sign into the Okta Admin dashboard to generate this value.

    7. Click Save changes.

    8. Done!


    Notes:

    IDP-initiated flows, SP-initiated flows, and Just In Time (JIT) provisioning are all supported.


    For SP-initiated Flows:

    • Open Lucidchart Sign in URL: https://www.lucidchart.com/saml/sso/[yourDomain].
