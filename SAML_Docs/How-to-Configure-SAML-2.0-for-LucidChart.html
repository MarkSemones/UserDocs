How to Configure SAML 2.0 for LucidChart

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Email LucidChart to enable SAML for your account.

  2. Copy the following text into the email and label it IDP Metadata. 

    Sign into the Okta Admin dashboard to generate this value.

  3. Copy and paste the following text into the message and label it IDP Issuer/Entity ID.

    Sign into the Okta Admin dashboard to generate this value.

  4. Copy and paste the following text into the message and label it Login URL/Sign on URL.

    Sign into the Okta Admin dashboard to generate this value.

  5. Copy and paste the following text into the message and label it Logout URL/Sign out URL.

    Sign into the Okta Admin dashboard to generate this value.

  6. After receiving confirmation from LucidChart that SAML is enabled, in the Okta Dashboard for the OfficeSpace Software application, select Okta username for the Default username format, as shown below. 

  7. Done!
