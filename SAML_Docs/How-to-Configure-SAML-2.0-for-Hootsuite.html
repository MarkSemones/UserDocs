How to Configure SAML 2.0 for Hootsuite

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Send an email message to Hootsuite and request that they enable SAML for your organization.
  2. Copy and paste the following in the SAML login URL into the message.

    Sign into the Okta Admin dashboard to generate this value.
  3. Select the link below to download a certificate. Attach the certificate to the message.

    Sign into the Okta Admin dashboard to generate this value.
  4. Hootsuite will process your request and send you an SSO Return URL.
  5. In Okta, go to the General tab for the Hootsuite app. Enter the SSO Return URL, and then select Save, as shown below.


  6. Done!
      • Just In Time (JIT) provisioning is not supported.
      • IdP-initiated and SP-initiated flows are supported. To use an SP-initiated flow, open https://hootsuite.com/sso, enter your email address, and select the Single Sign-On button.
