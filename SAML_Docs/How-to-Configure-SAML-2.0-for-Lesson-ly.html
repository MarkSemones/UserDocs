How to Configure SAML 2.0 for Lesson.ly

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Send an email to support@lesson.ly and request that they enable SAML 2.0 for your account.

  2. Attach the following information to the email:

    • SAML SSO URL: Copy and paste the following:

      Sign into the Okta Admin Dashboard to generate this variable.

    • Identity Provider Issuer: Copy and paste the following:

      Sign into the Okta Admin Dashboard to generate this variable.

    • x.509 Certificate: Download, then attach the following:

      Sign into the Okta Admin Dashboard to generate this variable.

    • Lesson.ly Account URL: https://[yourSubDomain].lesson.ly/

  3. Lesson.ly support will process your request. After receiving a confirmation email, you can start assigning people to the application.

  4. Done!

    Notes:

    IdP-initiated flows, SP-initiated flows, and Just In Time (JIT) provisioning are all supported.

    For SP-initiated flows:

    Go to https://[yourSubDomain].lesson.ly/auth/saml
