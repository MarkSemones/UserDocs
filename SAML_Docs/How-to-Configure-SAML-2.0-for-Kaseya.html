How to Configure SAML 2.0 for Kaseya

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Login to Kaseya as a user who has a Master Role.

  3. Navigate to AuthAnvil > Two Factor Auth > Configure Kaseya Logon.

  4. Scroll down to the Kaseya Single Sign On section at the bottom and do the following (see screen shot at end of step for reference):

    • Check the Enable Single Sign On to Kaseya box.

    • For Reply To URL, enter https://[sub-domain for the Kaseya server]/vsapres/web20/core/ssologin.aspx

    • Certificate: Download the following certificate, change the file extension from .cert to .cer, then upload to Kaseya.

      Sign into the Okta Admin Dashboard to generate this variable.

    • Click Save Settings.

  5. Done!
