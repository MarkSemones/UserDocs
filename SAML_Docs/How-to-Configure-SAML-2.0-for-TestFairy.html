How to Configure SAML 2.0 for TestFairy

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Log in to your TestFairy account.

  2. Open the Account Preferences page by selecting Preferences from the top right drop down menu:

  3. Select SAML/Single Sign-on from the left navigation pane.

  4. ID Provider Metadata: Copy and paste the following into this field:

    Sign in to Okta Admin app to have this variable generated for you.

  5. Click Update SAML ID Provider Metadata.

  6. To test, log out. You should now see the same login page, but with a new SSO LOGIN button:

  7. Done!

    Notes:

    SP-initiated flows and Just In Time (JIT) provisioning are supported.

    IdP-initiated flows are not supported.
