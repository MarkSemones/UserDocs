  2. In Okta, select the Sign On tab for the Lifesize app.

    • Click Edit to make the fields editable.

    • For the Encryption Certificate, click Browse to navigate to the lifesize.crt file you just saved, then click Upload to upload it to Okta.

    • Click Save when complete.

  3. Sign in to Lifesize Cloud with administrative privileges here: https://manage.lifesizecloud.com.

  4. Access Lifesize Cloud Advanced Settings by clicking the drop down button next to your name in the upper right corner of the page, then navigate to SSO Integration (see screen shot at end of step for reference):

    • In the SSO Configuration section:

      • Enter the following Identity Provider Issuer:

        Sign into the Okta Admin Dashboard to generate this variable.

      • Enter the following Login URL:

        Sign into the Okta Admin Dashboard to generate this variable.

      • Enter the following Certificate: 

        Sign into the Okta Admin Dashboard to generate this variable.

    • In the SAML Attribute Mapping section:

      • For First Name: Enter firstname.

      • For Last Name: Enter lastname.

      • For Email: Enter email.

    • Click Test.

  5. Once testing is successful, click Update to enable SSO.

  6. Copy the URL from the Relay State field:

  7. Back in Okta, select the Sign On tab for the Lifesize app.

    • Click Edit to make the fields editable.

    • Enter the Relay State you just copied into the Default Relay State field.

    • Click Save.

  8. Done!

    IdP-initiated flows, SP-initiated flows, and Just In Time (JIT) provisioning are all supported.
