How to Configure SAML 2.0 for Grovo

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

 

  1. Submit a request to Grovo to enable SAML 2.0 Single Sign-On for your account and provide the following information:
    • IDP Metadata:
       
    • Login URL/SignOn URL:Sign into the Okta Admin Dashboard to generate this variable.
  2. Grovo will notify you when the SAML settings have been added.
  3. Done!

 

Notes: SP-initiated flows and Just In Time (JIT) Provisioning are supported.  IDP-initiated flows are not supported.
