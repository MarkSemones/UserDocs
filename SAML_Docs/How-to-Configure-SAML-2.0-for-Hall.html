How to Configure SAML 2.0 for Hall

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Sign in to Hall with your credentials. 
  2. Navigate to Lobby > Settings > Single Sign On. The Single Sign-On Settings screen appears as shown below.

  3. On the Single Sign-On Settings screen:
    1. Check the Enabled box.

    2. Enter the following Login URL.

      Sign in to the Okta Admin app to generate this variable.

    3. Enter the following Logout URL.

      Sign in to the Okta Admin app to generate this variable.

    4. Enter the following Issuer URL.

      Sign in to the Okta Admin app to generate this variable.

    5. Enter the following Certificate Fingerprint.

      Sign in to the Okta Admin app to generate this variable.

    6. Select Save Settings.

  4. Done!
Note: IdP-initiated flows, SP-initiated flows, and Just In Time (JIT) provisioning are all supported.
  • To use the SP-initiated flow, use this URL: https://hall.com/networks/saml/<YourHallNetworkDomain>.
