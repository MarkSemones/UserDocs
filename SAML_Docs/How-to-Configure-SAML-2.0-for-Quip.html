How to Configure SAML 2.0 for Quip

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Copy and save the following IdP Metadata show below with the filename metadata.xml. 

  2. Send an email message to Quip and request that they enable SAML for your organization.

  3. Attach the metadata.xml file you created in step 1.
  4. Quip will process your request and notify you when SAML is enabled.
  5. After you receive the confirmation email from Quip, you can assign people to the application.
  6. Done!
Note: SP-initiated flows and Just In Time (JIT) provisioning are supported.
