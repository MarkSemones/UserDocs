      Sign in to the Okta Admin app to generate this variable.

    3. Copy and paste the following text into the message and label it Redirect Login URL.

      Sign in to the Okta Admin app to generate this variable.

    4. Copy the following IDP Metadata into a text file. Save it with the file name metadata.xml and attach it to the message.

      Sign into the Okta Admin dashboard to generate this value.
