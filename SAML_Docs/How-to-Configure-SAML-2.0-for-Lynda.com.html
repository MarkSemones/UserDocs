How to Configure SAML 2.0 for Lynda.com

Please sign-in to the Okta Admin app to have your organization specific variables generated for you. You might not be able to complete the setup without these generated variables.

  1. Send a message to Lynda.com and request that they enable SAML for your organization.

  2. Copy the following value into your message and label it Identity Provider Issuer/Entity Id.Sign into the Okta Admin dashboard to generate this value.

  3. Copy the following value into your message and label it Identity Provider HTTP POST URL.Sign into the Okta Admin dashboard to generate this value.

  4. Lynda.com will contact you when SAML is enabled.
  5. Done!
