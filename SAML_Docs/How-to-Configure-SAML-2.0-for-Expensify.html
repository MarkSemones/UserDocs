How to Configure SAML 2.0 for Expensify

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Generate the following IDP Metadata and save to a file named metadata.xml.

  2. Send an email to Expensify support team and request that they enable SAML 2.0 for your account.
  3. Attach the file metadata.xml that you created in step 1.

  4. Attach the following Login URL to the email.

  8. Expensify will process your request. After you receive a confirmation email, you can start assigning people to the application.

  9. Done!

    Notes:

    IDP-initiated flows and SP-initiated flows are supported. Just In Time (JIT) provisioning is not supported.

    For an SP-initated flow go to https://www.expensify.com/signin and select SAML. On the next page, enter your email address and click the Go button.
