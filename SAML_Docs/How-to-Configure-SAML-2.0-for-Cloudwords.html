How to Configure SAML 2.0 for Cloudwords

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Generate the following IDP Metadata and save to a file named metadata.xml.

     
  2. Send an email to Cloudwords Support and request that they enable SAML 2.0 for your account, attach the metadata.xml file you created in step 1, above.
  3. Cloudwords will process your request and after receiving a confirmation email from them, you can start assigning people to the application.

  4. Done!

    Notes:

    IDP-initiated and SP-initiated flows are supported.

    Just In Time (JIT) provisioning is not supported.
