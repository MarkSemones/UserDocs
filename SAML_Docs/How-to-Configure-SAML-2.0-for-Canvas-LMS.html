How to Configure SAML 2.0 for Canvas LMS

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Sign in to your Canvas LMS Admin Console, and click on your organization name in the upper left. 

  2. Select Authentication Settings.

  3. Copy and paste the following into the IdP Entity ID field: Sign into the Okta Admin dashboard to generate this value.

  4. Copy and paste the following in the Log On URL field: Sign into the Okta Admin dashboard to generate this value.
  5. Copy and paste the following in the Log Out URL field:Sign into the Okta Admin dashboard to generate this value.
  6. Copy and paste the following in the Change Password Link field. Make sure you replace  yourorg value with your Okta subdomain: https://yourorg.okta.com/user/settings/account

  7.  Copy and paste the following in the Certificate Fingerprint field:

    Sign into the Okta Admin dashboard to generate this value.

  8. Copy and paste the following in the Login Attribute field:NameID

  9. Copy and paste the following in the Identifier Format field:urn:oasis:names:tc:SAML:1.1:nameid-format:unspecified
  10. Save the settings.
  11. Done!
