How to Configure SAML 2.0 for SeamlessWeb

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Email SeamlessWeb client support to enable SAML 2.0 Single Sign-On for your account.

  2. Download and attach the following Certificate to the email.

    Sign into the Okta Admin dashboard to generate this value.

  3. SeamlessWeb client support will assign your company a firm code to sign in with SAML 2.0 Single Sign-On. You can assign SeamlessWeb access to your users at anytime from their user profile. Go to People page in Okta to view and edit user profiles.

  4. Done!
