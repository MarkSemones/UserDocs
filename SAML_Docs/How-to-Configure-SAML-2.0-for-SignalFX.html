How to Configure SAML 2.0 for SignalFX

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Sign into SignalFX as an administrator.

  2. Navigate to INTEGRATIONS > SAML SSO and select the NEW INTEGRATION button, as shown below.

  3. Refer to the screen shown below for steps 3a-3f.

    1. In the screen that opens, enter the following Name.

      Okta

    2. Save the information in the Integration ID field.
    3. Enter the following Public Key. 

      Sign in to the Okta Admin dashboard to generate this value.

    4. Enter the following Issuer URL.

      Sign in to the Okta Admin dashboard to generate this value.

    5. Enter the following Metadata URL.

      Sign in to the Okta Admin dashboard to generate this value.

    6. Click Save.
  4. In Okta, on the General tab for the SignalFX app, paste the Integration ID that you copied in step 3b, as shown below.

  5. Click Save.

  6. Done!

    Note: IdP-initiated flows and Just In Time (JIT) provisioning are supported. SP-initiated flows are not supported.
