How to Configure SAML 1.1 for Selectica

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

In order to use SAML 1.1 single sign-on to Selectica, you will need to supply the following information to Selectica and request that they enable SAML for your company's account.

  1. Your SAML Issuer ID:

    Log into Okta admin app to have this variable generated for you.

  2. Your Identity Provider Certificate:

    Log into Okta admin app to have this variable generated for you.

  3. Your Login Redirection URL:

    Log into Okta admin app to have this variable generated for you.
