This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. In Okta, select the General tab for the Changepoint app. Copy the Base URL, as shown below.

  2. in Okta, select the Sign On tab for the Changepoint app. Paste the URL you copied in step 1 to the Default Relay State field, as shown below and select Save.

  3. Contact the appropriate Changepoint administrator and send them the following two items.

    1. Copy and paste the following text into the message and label it Identity Provider Login URL.

      Sign in to the Okta Admin app to generate this variable.

    2. Select the following link to download the x509 Certificate. Then, attach it to the message.

      Sign in to the Okta Admin app to generate this variable.

  4. Done!

    Note: IdP-initiated and SP-initiated flows are supported. JIT provisioning is not supported.
