  2. On the Global Settings tab, click the edit... link to the right of the words Federated Authentication.

  3. Click the check box to the left of the words Enable Federated Authentication if it is not checked.

  4. Download the Identity Provider Certificate. 

    Sign into the Okta Admin dashboard to generate this value.
