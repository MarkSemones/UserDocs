How to Configure SAML 2.0 for Webcargo

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Sign in to Webcargo. Click your user name, and then select Settings. Note the URL that appears.

  2. On the left menu, select Team, as shown below.

  3. Select the SSO IdP tab, as shown below.

  4. Move the slider next to Okta to ON, as shown below.

  5. The window shown below appears on the right. 

  6. Copy the SAML Consumer URL shown in step 1. On the Okta Dashboard, go to the General tab of the Webcargo application and paste it in.

  7. Copy the following link to the Sign-on URL into step 2. Do not click View setup instruction again, as shown in step 2.

    Sign into the Okta Admin dashboard to generate this value.

  8. Copy the following Certificate in PEM text format and paste it into step 3. 

    Sign into the Okta Admin dashboard to generate this value.

  9. Select Save.

  10. On the Okta Dashboard for the Webcargo application, select Okta username for the Default username format, as shown below. 

  11. Done!
