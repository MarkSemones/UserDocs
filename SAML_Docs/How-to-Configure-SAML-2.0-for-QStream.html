How to Configure SAML 2.0 for QStream

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Email QStream support at support@qstream.com to enable SAML for your account.

  2. Copy the following Certificate Fingerprint and paste it into the email.

    Sign into the Okta Admin dashboard to generate this value.

  3. Copy the following Single Sign-on URL and paste it into the email.

    Sign into the Okta Admin dashboard to generate this value.

  4. Copy the following Issuer ID and paste it into the email.

    Sign into the Okta Admin dashboard to generate this value.

  5. QStream will notify you when SAML is enabled for your account. After SAML is enabled, you can use both IdP-initiated flow and SP-initiated flow.

    To use SP-initiated flow, go directly to the following URL and select Log in. Substitute your QStream domain for <your_subdomain>.

     https:/<your_subdomain>.qstream.com

  6. Done!
