This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Go to the Tinderbox administrator site for your Tinderbox domain, located at https://<yoursubomain>.mytinder.com/integrations/saml, where <yoursubdomain> is the subdomain name you entered on the Okta Tinderbox General tab, shown below.

  2. Select Integrations and go to the SAML Identity Provider Configuration screen, shown below.

  3. Check Enable SAML.
  4. Leave the field for IDP Start URL blank.
  5. Copy the following value to the  IDP SSO Target URL field.

    Sign into the Okta Admin dashboard to generate this value.
  6. Download the following x.509 Certificate.Sign into the Okta Admin dashboard to generate this value.
  7. Select Choose File and point to the certificate you just downloaded.

    Note: When you save the changes, the IDP Certificate Fingerprint field is populated.
  8. Keep the Name Identifier Format as urn:oasis:names:tc:SAML:1.1:nameid-format:emailAddress.
  9. Select Save Changes.
  10. Done!
