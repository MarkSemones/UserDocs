How to Configure SAML 2.0 for Yambla

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Email Yambla support to enable SAML for your account. 

  2. Copy the following IDP Metadata and paste it into the email: 

    Sign into the Okta Admin dashboard to generate this value.

  3. After receiving confirmation from Yambla that SAML is enabled, in the Okta Dashboard for the Yambla application, enter https://www.yambla.com/site/loginWithSAML into the Default Relay State field, as shown below.

  4. Select Okta username for the Default username format, as shown above.
  5. Select Save.
  6. To use SP initiated flow, go directly to the following URL. Note: The companyName is a unique value & you will have to replace this with your company name in Yambla
     https://www.yambla.com/<companyName>
  7. Done!
