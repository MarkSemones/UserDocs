How to Configure SAML 2.0 for Jama Software

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Send an email to Jama Software Support and request that they enable SAML 2.0 for your account.

  3. Ask Jama Software to use the following attribute mappings:

  4. Jama Software will process your request. After receiving a confirmation email, you can start assigning people to the application.

  5. Done!

    Notes:

    IDP-initiated flows, SP-initiated flows, and Just In Time (JIT) provisioning are all supported.

    For SP-initiated flows, go to the https://<yourSubDomain>.jamacloud.com/ page, where <yourSubDomain> is your Jama Software subdomain.
