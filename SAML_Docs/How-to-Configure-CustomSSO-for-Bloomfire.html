How to Configure SAML 2.0 for Bloomfire

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Go to the Settings page, scroll to the Miscellaneous section.

  2. Click New Key button if there is no API key present.

  3. Check the Enable API access checkbox.

  4. Copy your API Key and enter it in the Okta Single Sign-On section.

  5. Done!
