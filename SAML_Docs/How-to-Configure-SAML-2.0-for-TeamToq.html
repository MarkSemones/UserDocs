How to Configure SAML 2.0 for TeamToq

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Log into TeamToq as an administrator.
  2. Navigate to Admin page > SSO Administration:

  3. Scroll down and copy and save the value for Unique Company SSO URL.
  4. In Okta, select the General tab for the TeamToq app and enter the value just just saved in the Company SSO URL field:

  5. Click Save.
  6. On the Admin page > SSO Administration > Edit SSO Settings page, enter the following information (see screen capture at end of step for reference):

      • Click Create Token.
      • Enter a token name.
      • Click Create Token.
      • Take note of your token value, then click OK, got it.

      Sign into the Okta Admin Dashboard to generate this variable.
