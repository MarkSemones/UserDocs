Send an email to Mobi Wireless Management support and request SAML 2.0 access. Include the following four items.

  1. Copy the following text into the message and label it SAML Name ID Format.

    urn:oasis:names:tc:SAML:1.1:nameid-format:unspecified

  2. Copy the following URL into the message and label it SAML IDP SSO Target URL.

    Sign into the Okta Admin dashboard to generate this value.

  3. Download the following Identity Provider Certificate and attach it to the message.

    Sign into the Okta Admin dashboard to generate this value.

  4. Copy the following URL into the message and label it SAML SSO Logout Landing URL. 

    Sign into the Okta Admin dashboard to generate this value.
 Mobi Wireless Management support will contact you to confirm the setup.
