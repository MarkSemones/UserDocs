How to Configure SAML 2.0 for SugarCRM

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Sign into your SugarCRM instance as an Administrator.

  2. In the upper right corner, select Administrator > Admin. Then, select Users > Password Management.

  3. In the SAML Authentication section, check the Enable SAML Authentication box, as shown below.
