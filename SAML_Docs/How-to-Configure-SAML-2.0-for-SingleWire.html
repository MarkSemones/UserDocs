How to Configure SAML 2.0 for Singlewire InformaCast Mobile

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Save the following as saml-cert.pem:

    Sign into the Okta Admin Dashboard to generate this variable.
  2. In Okta, select the Sign On tab for the Singlewire InformaCast Mobile app, then click Edit.

    • For Encryption Certificate, click Browse to locate the saml-cert.pem certificate you saved earlier, then click Upload as shown below:

    • Click Save.

  3. Download the file by clicking on Identity Provider metadata, and then email the downloaded file to support@singlewire.com.

  4. The Singlewire InformaCast Mobile support team will process your request. After receiving a confirmation email, you can start assigning people to the application.

  5. Done!


Notes:

SP-initiated flows are supported.

IDP-initiated flows and Just In Time (JIT) provisioning are not supported.
