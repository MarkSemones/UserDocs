How to Configure SAML 2.0 for Dropbox

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Go to the Admin Console in Dropbox.

  2. Click on the Authentication tab.

  3. Check the Enable single sign-on checkbox.

  4. Check the Required checkbox.

  5. Copy the Okta Login URL into the Sign in URL field:

    Sign in to the Okta Admin app to have this variable generated for you.

  6. Download Okta certificate and load it into the X.509 Certificate field by clicking Change Certificate.

    Sign in to the Okta Admin app to have this variable generated for you

  7. Click Save Changes.

  8. Done!
