How to Configure SAML 2.0 for Kaleo

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Login to Kaleo as an administrator.
  2. Navigate to My Account > Tenant Panel screen.
  3. Click Settings in the left navigational pane, then select the Application Settings tab.
  4. Edit each of the following settings, and add the following values:

    • authentication.saml.idp_cert_fingerprint:Sign into the Okta Admin Dashboard to generate this variable.
    • authentication.saml.idp_sso_target_url:Sign into the Okta Admin Dashboard to generate this variable.


  5. Done!

    Notes:

    SP-initiated flows, IDP-initiated flows, and Just in Time (JIT) provisioning are all supported.
