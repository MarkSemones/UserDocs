How to Configure SAML 2.0 for LearnCore

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Email LearnCore Support and request that they enable SAML 2.0 Single Sign On for your account.

  2. Download the following Certificate and attach it to the email.

    Sign into the Okta Admin dashboard to generate this value.

  3. Copy and paste the following text into the message and label it IDP Issuer ID.

    Sign into the Okta Admin dashboard to generate this value.

  4. Copy and paste the following text into the message and label it Sign On URL.

    Sign into the Okta Admin dashboard to generate this value.

  5. Copy and paste the following text into the message and label it Sign Out URL.

    Sign into the Okta Admin dashboard to generate this value.
  6. Include your domain in the mail and label it Domain.
  7. LearnCore will notify you when SAML is enabled.

  8. Done!

    Note: IdP-initiated flows, SP-initiated flows, and Just In Time (JIT) provisioning are all supported.
