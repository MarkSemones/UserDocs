How to Configure SAML 2.0 for Symmetryk

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Sign into Symmetryk. Before setting up SAML, you must create a user.

  2. Click the Administration chicklet, as shown below.

  3. Select Admistration from the menu on the left side of the screen. Then, select Add User in the screen that opens on the right, as shown below.

  4. Complete and save the information for the new user.

  5. Select Settings from the menu on the left side of the screen. Then, select Add SAML provider, as shown below.

  6. In the Settings window that opens, copy and paste the following IDP Metadata, as shown below. When done, click Save.

    Sign into the Okta Admin dashboard to generate this value.

  7. After adding the SAML provider, activate it by clicking the activation switch to move it to the right, as shown below.

  8. Done!

    Note: SP-initiated flows and IdP-initiated flows are supported. Just In Time (JIT) provisioning are not supported.
