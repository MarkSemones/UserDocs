    Notes:

    IdP-initiated flows and SP-initiated flows are supported.

    Just In Time (JIT) provisioning is not supported.

    The application only works through native iPass Open Mobile client.

    To install the iPass Open Mobile client:

    1. Download the iPass Open Mobile client from http://www.ipass.com/download-mobile-client/.
    2. Use the SSO credentials to activate your service.
